# CCTV Viewer

A simple and minimal camera viewer web app with two modes:

- Full Mode – grid of camera thumbnails (click any tile to open it)
- List Mode – large player with a scrollable list of cameras

Built entirely in HTML, CSS, and JavaScript — no installation required.

---

## Features

- Toggle between Full / List modes with `M`
- View single camera with location label
- Clickable camera tiles
- Keyboard navigation (`←`, `→`, `M`, `F`)
- Works locally or via GitHub Pages
- Remembers last used view

---

## System Requirements

- Any modern browser (Chrome, Edge, Firefox, Safari)
- Optional: Python, Node.js, or VS Code for local server
